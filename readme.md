# 出席生徒確認用顔認証システム（予定）　（データベース作成途中まで）

## DEMO

  - デプロイしている場合はURLを記入（任意）

## 紹介と使い方

  - 生徒の名前・顔写真を登録し、データベースを作成

  - 写真（動画）から顔認証を行い。

## 工夫した点

  - 学習したphpによるテキストの保存から、クラスごとの自動選別　⇨ ソートで番号順に並べ替えを行う。
  - 画像データの保存を行う。もう少しで表示までできそう。
